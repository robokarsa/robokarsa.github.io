Sekilas Cerita Tentang Kami

Mengapa kami ada?

Jumlah sekolah di pulau Jawa? Banyak. Jumlah layanan kursus robotik di pulau Jawa? Banyak juga. Sementara itu, jumlah sekolah di pulau NTB? Banyak. Sedangkan, jumlah layanan kursus robotik di pulau NTB? Tidak ada.

Oleh karena itulah RoboKarsa hadir di NTB.

  • Biaya terjangkau.
  • Beragam project.
  • Pilihan materi yang fleksibel.
  • Materi lebih mudah dipahami.

RoboKarsa hadir untuk mewadahi anak yang membutuhkan kegiatan positif di luar akademik sekolah namun disaat yang bersaamaan, juga membantu mengembangkan dan mengasah logika dan hardskill anak di bidang teknologi.

Mengapa memilih RoboKarsa?

Di Indonesia, kami hanyalah satu dari banyak tempat yang menawarkan kursus Robotik dan pemrograman untuk anak. Namun di NTB, terutama di Kota Mataram, kami menjadi satu-satunya tempat yang menyediakan arena bermain bagi anak yang ingin menyalurkan bakatnya di bidang teknologi.

Biaya Terjangkau

Dengan penyesuaian yang dilakukan oleh RoboKarsa, siswa dapat belajar dengan leluasa, orangtua tidak perlu risau perihal biaya.

Beragam Project

Beragam project yang tersedia di RoboKarsa membuat siswa tidak akan mudah bosan dan jenuh ketika belajar.

Mudah Dimengerti

Siswa tak harus bergelar S1 terlebih dahulu karena materi yang dibawakan akan disesuaikan dengan tingkat pemahaman siswa.

Level

Newbie

Instalasi dan Mengenal Arduino IDE

Arduino IDE merupakan software yang akan kita gunakan di laptop dan fungsi utamanya adalah untuk menulis program-program yang akan memerintah robot serta menampilkan output data dari robot atau sensor-sensor.

#00_NEWBIE

Mengenal Board Arduino Nano

Pelajaran ini akan mengenalkan Siswa kepada Arduino Nano yang akan digunakan sebagai otak dari robot serta tempat menyimpan program yang telah ditulis dalam bahasa C++ (dibaca C Plus Plus) sehingga robot bisa beroperasi sesuai dengan perintah.

#00_NEWBIE

Mengedipkan Lampu LED

Menulis program dalam bahasa C++ untuk memerintahkan Arduino Nano agar menyalakan dan mematikan lampu LED pada Arduino Nano adalah pelajaran paling mendasar yang harus dikuasai oleh siswa sebelum melangkah ke pelajaran berikutnya.

#02_NEWBIE

Modul Lampu Lalu Lintas Mini

Setelah mempelajari bagaimana mengedipkan lampu LED pada Arduino Nano, kini saatnya mempelajari bagaimana mengedipkan lampu LED pada Modul Lalu Lintas Mini dan memprogram kapan lampu merah menyala dan kapan lampu hijau menyala.

#03_NEWBIE

Mengenal Breadboard

Komponen yang memiliki fungsi utama sebagai pengganti kabel dan untuk memperbanyak pin-pin daya akan sering kita gunakan kedepannya baik untuk keperluan prototiping project ataupun mempelajari sensor-sensor yang tersedia di RoboKarsa.

#04_NEWBIE

Serial Monitor Arduino IDE

Serial Monitor merupakaan tools yang terdapat pada Arduino IDE agar kita bisa berkomunikasi dengan Arduino Nano melalui laptop seperti menampilkan teks Hello World ataupun menampilkan nilai hasil pembacaan sensor.

#05_NEWBIE

Level

Beginner

Modul Sensor Cahaya

Level Beginner akan berfokus untuk mengenalkan Siswa dengan komponen-komponen sensor sembari mengenalkan Siswa kepada perintah serta fungsi-fungsi yang bisa dibuat dalam bahasa C++ secara perlahan.

#07_BEGINNER

Modul Sensor Jarak Ultrasonik

Karena prinsip kerjanya yang menggunakan lama durasi pantulan suara Ultrasonik, sensor ini digunakan oleh robot karena memiliki kemampuan untuk mengukur jarak dengan sehingga robot bisa mendeteksi keberadaan benda di depannya.

#08_BEGINNER

Modul Sensor Halangan Infra Merah

Modul sensor ini mirip dengan Modul Sensor Jarak Ultrasonik yang menggunakan prinsip pantulan untuk mendeteksi objek di depannya. Namun modul sensor ini menggunakan pantulan cahaya Infra Merah dan tidak bisa mendeteksi jarak dengan pasti.

#09_BEGINNER

Modul Sensor Api

Modul sensor api dapat digunakan sebagai mata robot untuk mendeteksi keberadaan api. Modul sensor api pada dasarnya bekerja mendeteksi cahaya infra merah yang dipancarkan oleh api dan dapat digunakan untuk mengetahui intensitas api.

#10_BEGINNER

Modul Limit Switch

Limit Switch digunakan untuk memutuskan atau menghubungkan suatu rangkaian. Pada pelajaran ini Siswa akan mempelajari bagaimana memprogramnya serta belajar menggunakan solder untuk pertama kalinya.

#11_BEGINNER

Modul Sensor Suara

Sejauh ini, kita sudah membahas sensor-sensor yang bisa kita gunakan sebagai mata untuk robot. Modul sensor kali ini akan digunakan sebagai telinga untuk robot, yakni menangkap suara yang ada di sekitar robot.

#12_BEGINNER

Level

Intermediate

Light Sensor dan Modul LED

Pada level ini, Siswa akan belajar bagaimana memprogram dua Modul yang berbeda sekaligus sehingga mampu memperdalam pemahaman Siswa mengenai bahasa pemrograman C++.

#13_INTERMEDIATE

Distance Sensor dan Modul LED

Menampilkan nilai jarak yang dibaca sensor pada Serial Monitor serta memberi indikator dengan ketiga lampu pada Modul LED untuk tiga jarak yang berbeda adalah target utama pada pelajaran ini.

#14_INTERMEDIATE

Proximity Sensor dan Modul LED

Tentu, Proximity Sensor merupakan nama lain dari Modul Sensor Halangan Infra Merah pada level Beginner sebelumnya. Kali ini belajar bagaimana memprogram Proximity Sensor bersama Modul LED.

#15_INTERMEDIATE

Flame Sensor dan Modul LED

Api kalau kecil menjadi kawan, kalau besar menjadi lawan. Disini kita akan memprogram lebih dalam Fire Sensor bersama Modul LED untuk menyalakan lampu sesuai intensitas api yang menyala.

#16_INTERMEDIATE

Limit Switch dan Modul LED

Selalu ada banyak cara memanfaatkan Modul LED ini untuk belajar. Pada pelajaran ini, kita akan membuat program mini game pertama kita dengan Modul LED dan Limit Switch.

#17_INTERMEDIATE

Sound Sensor dan Modul LED

Jangan bosan dengan Modul LED ya, karena di pelajaran kali ini kita akan mengkombinasikan Sound Sensor dengan Modul LED untuk membuat Sound Meter, yakni pengukur intensitas suara.

#18_INTERMEDIATE

Level

Advanced

Modul Servo SG90

Manusia memiliki otot untuk menggerakkan tulang dan badannya kesana kemari ataupun mengangkat beban berat. Untuk robot, robot menggunakan Servo Motor SG90 untuk menggerakkan suatu bagian pada tubuhnya.

#19_ADVANCED

Modul Servo SG90 dan Potentiometer

Jika sebelumnya kita sudah belajar bagaimana mengontrol Modul Servo melalui perintah pada program, pada pelajaran ini akan mengkombinasikan Modul Servo dengan Potensiometer untuk mengubah posisi sudutnya.

#20_ADVANCED

Modul L298N dan DC Motor

Anggaplah DC Motor sebagai kaki yang bisa membantu robot bergerak dengan cepat dan Modul L298N adalah otak tambahan untuk mengontrol DC Motor agar bisa berputar baik berlawanan arah jarum jam atau searah jarum jam.

#21_ADVANCED

spL298N, DC Motor, dan Proximity Sensor

Pada pelajaran ini akan memahami bagaimana mengkombinasikan Proximity Sensor sebagai pengendali arah gerak dari DC Motor dimana DC Motor akan berputar ketika ada tangan di depannya dan diam ketika tidak ada tangan di depannya.

#22_ADVANCED

Fade LED_AnalogWrite() Function

Kita biasa menggunakan perintah digitalWrite() untuk menyalakan atau mematikan lampu. Pada pelajaran ini kita akan memahami bagaimana menggunakan perintah analogWrite() untuk mengatur tingkat kecerahan cahaya pada Modul LED.

#23_ADVANCED

Switch Case LED Control

Setelah terbiasa dengan fungsi If Else untuk mengontrol suatu kondisi, kini saatnya Siswa memahami fungsi baru bagaimana mengontrol LED dengan fungsi Switch Case untuk beberapa kondisi yang berbeda.

#24_ADVANCED

Level

Expert

Laser Shooting Game

Level ini difokuskan untuk mengajak Siswa mengaplikasikan pelajaran yang telah dipelajari pada level sebelumnya dengan membuat project. Project ini akan menembakkan cahaya laser ke Sensor Cahaya dan Modul LED akan menghitung jumlah tembakannya.

#26_EXPERT

Multiplayer Laser Shooting Game

Jika sebelumnya project Laser Shooting Game menuntut kita untuk melawan waktu kita sendiri, kali ini akan membuat Laser Shooting yang Game yang bisa dimainkan oleh dua orang dan beradu cepat untuk mematikan lampu pada Modul LED.

#27_EXPERT

Light Follower Robot

Kali ini akan memanfaatkan kembali Sensor Cahaya dari project sebelumnya sebagai mata untuk robot. Robot yang dibuat oleh Siswa akan bergerak mengikuti sumber cahaya senter yang menyorot Sensor Cahaya. Namun akan diam apabila tidak disorot oleh cahaya senter.

#28_EXPERT

Line Follower Robot

Masih ingat fungsi Sensor Halangan atau Proximity Sensor? Sensor tersebut juga dapat digunakan untuk mendeteksi keberadaan garis hitam di atas permukaan putih yang menjadi jalur agar robot bisa bergerak sesuai lintasan.

#29_EXPERT

Rock Paper Scissor Game

Benar, semua orang pasti tahu permainan batu kertas gunting. Bermain batu kertas gunting melawan manusia mungkin sepele, cobalah menantang diri dengan bermain batu kertas gunting melawan robot dan lihat seberapa hebat kita melawan robot.

#30_EXPERT

Obstacle Avoider Robot

Project ini memanfaatkan Sensor Ultrasonik sebagai mata robot untuk mendeteksi keberadaan benda di depannya. Sehingga ketika ada benda di depannya, robot akan bergerak mundur dan berbelok ke arah kanan atau kiri.

#31_EXPERT
